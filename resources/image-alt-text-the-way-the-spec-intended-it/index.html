Image Alt Text, the Way the Spec Intended It
In HTML, you can give a text description to images, called the alt text . For the sake of accessibility (i.e. people who cannot see the image or for whom it takes a long time to download), you should always :
  • Provide alt text
  • Or explicitly declare no alt text (for purely decorative images)
